MaskIt v1.0 - jQuery plugin

I've created this plugin for fun and simply to sharpen my jQuery plugin development skills. Feel free to use it in your projects, modify, change and do what you will, as long as you keep the copyright note at the top of plugin code. Also if you do something cool with it I would appreciate if you let me know. I would love to put link to it on this site then.

Compatibility

I tested it briefly with latest Firefox, Chrome, IE7/8/9 on Windows 7, and iPad3 with IOS6. If you find any bugs however, please do let me know on via my blog site http://spirytoos.blogspot.com.

How to use?

Download js file

jquery.maskit1.0.js (with comments if you dare, ~22kb)

jquery.maskit1.0.min.js (compressed, ~5.5kb)

Download jQuery (or link to it directly) before using plugin. Then link to plugin, sample code below:

<script type="text/javascript" src="jquery.min.js"></script>
<script src="jquery.maskit1.0.min.js"></script>

When specifying special characters MaskIt uses regular expressions (regex., see examples below). Regex can be complicated and very powerful, however if you not familiar with it I would suggest to Google it as there are heaps of resources on the subject. For those who do not know how to specify regex patterns, few very simple examples below:

[a-c] - matches any of these characters: abcd
[abc] - matches any of these characters: abc
[a-cA-Z] - matches any of these characters: abc and ABCDE....Z all the way till Z (uppercase)
[1-5o-s] - matches any of these characters: 12345 as well as matches lowercase opqrs

Examples

$("#date1").maskFields("dd/dd/dddd");
$("#withPh").maskFields("dd/dd/dddd" , { "alwaysDisplayPlaceholder": false });
$("#dash").maskFields("dd/dd/dddd", { "placeholderChar" : "-"});
$("#date2").maskFields("dd/aaa/dddd");
$("#date3").maskFields("dd/bd/20rd", {"specialChars" : { "b" : "[01]","r" : "[12]" }});
$("#phone1").maskFields("(dd) dddd-dddd");
$("#phone2").maskFields("(dd) dddd-dddd");
$("#phone3").maskFields("(dd) dddd-dddd ?ext ddd");
$("#time1").maskFields("dd:dd:dd");
$("#time2").maskFields("dd:dd?:dd");
$("#time4").maskFields("hd:ed?:ed lm",{"specialChars" : { "h" : "[012]","e" : "[0-5]","l" : "[ap]","m" : "m" }});
$("#hex1").maskFields("#dddddd");
$("#hex2").maskFields("#hhh?hhh",{"specialChars" : { "h" : "[0-9a-f]" }});
$("#money1").maskFields("$dd.dd");
$("#money2").maskFields("$dd?.dd");
$("#money3").maskFields("$vd?.9c",{"specialChars" : { "v" : "[23]","c" : "[59]" }});
$("#cc1").maskFields("dddd dddd dddd dddd");
$("#cc2").maskFields("dddd dddd dddd dddd ?(option^al) ddd");
$("#sep2").maskFields("Enter ^d^ay dd, month dd, ye^ar dddd"); $("#sep3").maskFields("Enter day dd, month dd, year dddd");
$("#sep4").maskFields("Enter ^d^ay dd, month dd?, ye^ar dddd");
$("#sep1").maskFields("dd\\dd\\dd");
$("#sep5").maskFields("?ddd aaa *^**");
$("#callback1").maskFields("dd/dd/20dd", {"onCompleted": function(e,t) { alert("e.keyCode:"+e.keyCode+" Entered: "+$(t).val()); }});
$("#callback2").maskFields("bd/dd/20dd", {"specialChars" : { "b" : "[01]" }, "onKeyDown": function(e,t) { return false; }});
